Stripe Payment Implementation

  • In index.html paste the stripe script i.e refer to index.html file
  • Then paste the secret key in environment file
  • Then declare a variable name stipe in ts file
  • Create the form in html file with id or template reference using div rather than input
  • On ngAfterViewInit lifecycle hook declare the card styling options, and assign the secret key to stripe.
  • Declare stripe elements
  • Then create the card with create function with pass card as stripe card variable and mount it.
  • Then on submit call the createToken method to create the token and the result is an object.
  • createToken method is a promise with either resolve or reject
  • on resolve onsuccess function is called and on reject onerror method is called.
